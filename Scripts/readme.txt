
- Preprocessing: Holds all scripts used to preprocess raw RNA sequencing data, input files to scripts and a readme file with information on the scripts.

- Quantification: Holds all scripts used to quantify processed RNA sequencing data to obtain gene counts, input files to scripts and a readme file with information on the scripts.

- Transcript_Gene_Mapping: Holds scripts that map transcripts to gene IDs, their input files and a readme file with information on the scripts.
